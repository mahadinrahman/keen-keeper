################################################################

** ProjectName: KeenKeeper

************************************************************

**Description:

This is a React-based friendship management application designed to help users maintain meaningful relationships. The app displays a list of friends, tracks communication activities, and reminds users when to reconnect. Users can view detailed friend profiles, perform quick interactions like call, text, or video, and monitor communication history through a Timeline. The Stats section visualizes interaction data using charts, while global state management with Context API keeps activity data synchronized across pages. Dynamic routing, filtering, and responsive UI provide a smooth and organized experience for managing personal connections.

*******************************************************************

**Technologies:

React.js,React Router DOM,Tailwind CSS,Daisi UI,Recharts

***************************************************************

**Key Features:

##Friend Management & Details View:
Users can browse a list of friends, view detailed profiles, and see important information like last contact, goals, and next due date.

##Interaction Timeline with Filtering:
The Timeline tracks communication activities (Call, Text, Video) and allows filtering by interaction type for better organization.

##Analytics Dashboard:
A visual Stats section displays interaction data using charts, helping users understand their communication patterns.
